# Qi

Headless CMS created on top of LEMP stack (https://github.com/chonla/lemp).

## Current status

In progress

## Start containers

```
docker-compose up -d
```

## Stop containers

```
docker-compose down
```

## What does Qi provide?

* Users
* Posts
